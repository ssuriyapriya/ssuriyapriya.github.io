Project Overview

Medical reports contain vast amounts of unstructured text that are time-consuming for clinicians to review. This project developed an NLP system to automatically generate concise summaries of medical reports, retaining critical clinical entities like diagnosis and treatment plans.

Key Methodology

  • Architecture: Utilized Bidirectional LSTM (Bi-LSTM) networks with Attention mechanisms to handle the sequential nature of clinical text.
  • Preprocessing: Implemented extensive text cleaning, tokenization, and stop-word removal specific to medical ontology.
  • Evaluation: Assessed model performance using ROUGE scores to ensure summaries accurately reflected the source text.
