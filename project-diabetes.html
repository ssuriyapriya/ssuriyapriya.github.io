Diabetes Readmission Risk Assessment System

Decision Support System to Predict 30-Day Hospital Readmissions

Machine Learning Python Healthcare Analytics XGBoost
Project Overview

Hospital readmissions for diabetic patients are a major cost driver and quality indicator for healthcare systems. This project focused on developing a Clinical Decision Support System (CDSS) to predict the likelihood of a patient being readmitted within 30 days of discharge.

Key Methodologies
  • Data Preprocessing: Handled class imbalance using SMOTEENN (combining oversampling and undersampling) and imputed missing clinical values.
  • Feature Engineering: Created new features capturing temporal patterns and comorbidity interactions to improve model sensitivity.
  • Modeling: Trained a Gradient Boosting (XGBoost) model which outperformed Logistic Regression and Random Forest.
Results & Impact
  • Achieved a predictive accuracy of 94.7%.
  • Identified key predictors of readmission through sensitivity analysis, providing actionable insights for care management protocols.
  • Demonstrated that a 1-day reduction in hospital stay was associated with a 2.1% increase in readmission risk for high-risk cohorts.
