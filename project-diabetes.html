Project Overview

The opioid crisis is a critical public health challenge in Allegheny County. This project aimed to scientifically determine the optimal locations for new Safe Consumption Sites (SCS) to maximize life-saving coverage. We utilized a "Predict-Then-Optimize" framework to forecast future overdose events and place resources where they are needed most.

Methodology

  • Prediction: Developed 106 individual ARIMA time-series models, one for every ZIP code in the county, to accurately forecast fatal overdose events.
  • Optimization: Formulated a Maximal Covering Location Problem (MCLP) using Gurobi optimization. This mathematically selected site locations that covered the highest number of predicted overdose events under budget constraints.

Impact

The optimized model demonstrated a potential to increase network service coverage by 10 percentage points compared to current ad-hoc placement strategies. This corresponds to covering an additional 50 forecasted deaths per year, providing a robust, data-driven argument for policy changes.
