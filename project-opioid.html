Opioid Crisis Intervention

A Predict-Then-Optimize Framework for Safe Consumption Sites
Fall 2025 Python, Gurobi, ARIMA Role: Project Lead

Executive Summary & Context

Allegheny County is facing a severe public health crisis driven by opioid overdoses. Fatal accidental overdoses have risen by 107% between 2015 and 2017, outpacing national averages. To mitigate this, harm reduction strategies such as Safe Consumption Sites (SCS) are critical.

The Challenge: Establishing an SCS is resource-intensive. With a limited budget, where should a new site be placed to maximize its life-saving potential?

This project employs a "Predict-Then-Optimize" framework. We first forecast future overdose events by ZIP code and then use mathematical optimization to select the ideal site location that covers the maximum number of predicted incidents.

Phase 1: Predicting Overdose Events

We utilized historical overdose data from the Western Pennsylvania Regional Data Center (WPRDC). To capture localized trends, we moved beyond a county-wide model and developed granular forecasts.

Methodology:

  • Data Preparation: Aggregated 10 years of fatal overdose data into monthly time series for each of the 106 ZIP codes in Allegheny County.
  • Modeling: Trained 106 individual ARIMA (AutoRegressive Integrated Moving Average) models. Parameters (p,d,q) were optimized for each ZIP code using grid search based on AIC scores.
  • Validation: The models were validated on the most recent 12 months of data to ensure they could accurately capture seasonal spikes and rising trends.

Phase 2: Optimization (MCLP)

With forecasts in hand, the next step was resource allocation. We formulated the problem as a Maximal Covering Location Problem (MCLP).

Optimization Constraints:

  • Service Radius: We defined "coverage" as a 10-minute driving distance or reasonable walking distance, calculated using the Haversine formula to determine proximity between ZIP code centroids.
  • Objective Function: Maximize the total number of predicted fatal overdoses covered by the new site location.
  • Solver: Implemented the linear programming model using the Gurobi Optimizer in Python.
Why Gurobi? It allows for finding the globally optimal solution efficiently, ensuring we don't settle for a sub-optimal location based on heuristics.

Results & Strategic Impact

The optimization model identified McKeesport (ZIP 15133) as the optimal location for a new Safe Consumption Site. This area is currently an overdose hotspot with zero existing coverage.

Quantifiable Outcomes:

  • Coverage Increase: The new site would increase the total network coverage of fatal overdoses from 23% to 33% (a 10 percentage point jump).
  • Lives Impacted: Based on our forecasts, this single site would cover an additional 50 predicted fatal overdoses annually.
  • Policy Recommendation: We recommended a "Hub-and-Spoke" model, with Prevention Point Pittsburgh (PPP) as the central hub and a new satellite facility in McKeesport to serve the Mon Valley region.
